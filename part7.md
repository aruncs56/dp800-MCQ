## DP-800 AI & Vector Search Cheat Sheet

### Core Concepts

| Concept       | Meaning                               |
| ------------- | ------------------------------------- |
| Embedding     | Numeric representation of data        |
| Vector        | Array of numbers representing meaning |
| Vector Search | Similarity-based search               |
| ANN           | Fast approximate search               |
| ENN           | Exact similarity search               |
| RAG           | Retrieval + AI generation             |
| Chunking      | Splitting text into smaller parts     |
| Hybrid Search | Keyword + vector search               |
| RRF           | Combines ranking results              |

---

### Key Exam Patterns

* “semantic meaning” → Vector search / embeddings
* “exact match” → relational SQL
* “best performance vs accuracy tradeoff” → ANN
* “combine AI with external documents” → RAG
* “improve relevance using both keyword + AI” → Hybrid search
